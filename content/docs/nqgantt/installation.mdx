description: Add @nqlib/nqgantt to a React app.

```bash
pnpm add @nqlib/nqgantt
```

### Local iteration in this showcase

```bash
pnpm nqgantt:status
pnpm nqgantt:sync-lib   # refresh bar UI reference copies
pnpm dev:local          # USE_LOCAL_NQGANTT when iterating on engine
```

See `NQGANTT-WORKSPACE.md` for bar UI sync and release notes.
