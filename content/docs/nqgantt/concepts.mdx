nqgantt splits **engine** (schedule math) from **bar UI**. This app wires both in roadmap demos.

| Concept | Where |
|---------|--------|
| Roadmap / timeline demo | `src/nqgantt/demos/` |
| Bar UI reference copies | `src/nqgantt/lib/` (sync with `pnpm nqgantt:sync-lib`) |
| Theme overrides | `src/nqgantt/gantt-theme.css` |

For board views over the same work set, see [nqgrid](/docs/nqgrid). For surrounding chrome, see [nqui](/docs/nqui).
