```bash
pnpm add @nqlib/nqgrid
```

### Local iteration in this showcase

```bash
pnpm nqgrid:status
pnpm dev:local   # USE_LOCAL_NQGRID → ../nqgrid/src
```

See `NQGRID-WORKSPACE.md` in the showcase repo for publish and alias notes.
