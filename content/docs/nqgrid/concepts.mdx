nqgrid is a **headless** engine. This app mounts it in product-shaped demos rather than a component API catalog.

| Concept | Live surface |
|---------|--------------|
| Spreadsheet / pivot | Spreadsheet demo in this app (`src/nqgrid/demos/spreadsheet/`) |
| Projects board | Projects demo (`src/nqgrid/demos/projects/`) |
| Column models | Field panels and schema editors in those demos |

For chrome around the grid, use [nqui](/docs/nqui). For timelines over the same work items, see [nqgantt](/docs/nqgantt).
